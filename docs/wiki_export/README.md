# Exportación del Wiki

No se encontraron páginas en el wiki público del repositorio `Juaker1/Proyecto-toxinas` (URL: <https://github.com/Juaker1/Proyecto-toxinas/wiki>).

Si el proyecto usa DeepWiki u otra documentación externa y deseas que la exporte:

- Comparte el origen (por ejemplo, URL de DeepWiki, credenciales si las requiere, o un dump/export existente).
- Alternativamente, habilita el Wiki en GitHub y agrega páginas; luego las vuelvo a exportar aquí como Markdown.
